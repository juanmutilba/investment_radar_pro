RADAR QUANTAMENTAL v2.0 - ESTRUCTURA MODULAR

1. OBJETIVO
Sistema quantamental modular orientado a detectar:
- oportunidades de compra
- activos en seguimiento
- deterioro técnico
- cambios respecto del último radar previo
- prioridad operativa diaria
- lectura descriptiva por empresa y sector
- perfil de riesgo por beta
- fuerza relativa por sector
- top 10 oportunidades del día
- separación entre radar USA/CEDEAR y radar Argentina

2. ESTRUCTURA DE ARCHIVOS
- main.py -> orquestador principal
- config.py -> parámetros globales
- engines/usa_engine.py -> motor USA / CEDEAR subyacente
- engines/argentina_engine.py -> motor local argentino
- core/technicals.py -> indicadores técnicos
- core/scoring.py -> score técnico y fundamental
- core/risk.py -> perfil de riesgo y risk score
- core/signals.py -> setup, señales, convicción, prioridad, evolución
- export/exporter.py -> exportación Excel y CSV
- export/excel_format.py -> formato visual, semáforos y comentarios
- data/universe_usa.py -> universo USA
- data/universe_arg.py -> universo Argentina
- utils/history.py -> comparación histórica

4. SCORES DEL MODELO
4.1 TechScore
Incluye únicamente indicadores técnicos:
- RSI < 30 => +3
- RSI entre 30 y 40 => +1
- Precio < MA50 => +1
- MA50 > MA200 => +1
- MACD line > Signal line => +1

Puntaje máximo: 7

4.2 FundScore
Incluye únicamente variables fundamentales / valuación:
- NetIncome > 0 => +1
- EBITDA > 0 => +1
- DebtToEquity < 200 => +1
- PE < 15 => +1
- Upside > 5% => +1

Puntaje máximo: 5

4.3 RiskScore
Incluye únicamente beta:
- Beta entre 0.8 y 1.6 => +1
- Beta entre 0.6 y 2.0 => 0
- Beta fuera de ese rango => -1
- Sin dato => 0

Rango: -1 a +1

4.4 Score total
TotalScore = TechScore + FundScore + RiskScore
Máximo actual: 13

17. COMENTARIOS EN EXCEL
Se agregan comentarios en los encabezados:
- TechScore -> Indicadores técnicos. Máximo 7 puntos.
- FundScore -> Resumen fundamental. Máximo 5 puntos.
- RiskScore -> Beta. Máximo 1, mínimo -1.
- TotalScore -> Suma de TechScore + FundScore + RiskScore.
- SignalState -> Clasificación operativa automática.
- PrioridadRadar -> Prioridad diaria de seguimiento.

18. SEMÁFOROS EN EXCEL
18.1 SignalState
- COMPRA PRIORITARIA => verde fuerte
- COMPRA POTENCIAL => verde claro
- SEGUIMIENTO => amarillo
- TOMA DE GANANCIA => naranja
- SOBREEXTENDIDA => rojo
- DEBILITÁNDOSE => rojo
- EVITAR => gris

18.2 Evolucion
- MEJORANDO => verde claro
- DETERIORANDO => rojo

- SIN CAMBIOS => gris
- SIN HISTORIAL => gris

18.3 TotalScore
- >= 10 => verde fuerte
- 8 a 9 => verde claro
- 5 a 7 => amarillo
- < 5 => rojo

18.4 CambioScore
- >= 2 => verde fuerte
- > 0 => verde claro
- = 0 => gris
- -1 => naranja
- <= -2 => rojo

18.5 PrioridadRadar
- ALTA => verde fuerte
- MEDIA => verde claro
- BAJA => amarillo
- IGNORAR => gris

18.6 RiskProfile
- BALANCEADO => verde claro
- AGRESIVO => amarillo
- DEFENSIVO => gris
- ESPECULATIVO => rojo

20. CRITERIO SOBRE CEDEARS
El radar USA analiza el subyacente en USD.
No mezcla precio local del CEDEAR con fundamentos en USD.
Las especies tipo AAPLC / AAPLD no forman parte del motor base.
